도서관 전산화 대학원 수업 프로젝트

1. 프로젝트 개요

본 프로젝트는 '여행과 우리'라는 교육과정을 디지털 환경으로 구축하여, 학습자들이 도서관의 정보 자원을 전산화된 시스템 안에서 어떻게 활용하고 학습할 수 있는지를 구현한 결과물입니다.

HTML5 XML CSS3 JavaScript Python Turtle

2. 역할 분담 및 주요 내용

구성원 A

제 1단원 & 학습 자료

여행의 시작과 기초 학습 자료 수집. 웹페이지의 기본 뼈대를 구성하고 학습 가이드라인 제시.

구성원 B

제 2, 3단원

본격적인 여행 테마 전개. 데이터의 구조적 표현(XML)과 시각적 디자인(CSS)을 적용한 콘텐츠 제작.

구성원 C

제 4단원

교육과정 마무리 및 평가. JavaScript를 이용한 동적 상호작용 및 Turtle 활용 시각 요소 도입.

구성원 D

전체 시스템 통합

각 단원별 파일 병합 및 GitHub 호스팅. 전체 UI/UX 통일감 조정 및 메인 페이지 제작.

3. 수업을 통한 핵심 학습 역량

  • 정보의 구조화: XML과 HTML을 통해 교육 콘텐츠를 체계적으로 분류하고 배치하는 능력 배양.
  • 프로그래밍 사고: JavaScript와 Turtle을 활용하여 단순 정적 페이지를 넘어선 인터랙티브 요소 구현.
  • 협업 시스템: 개별적인 작업물을 하나의 완성된 웹 서비스로 통합하는 전산화 프로세스 이해.
